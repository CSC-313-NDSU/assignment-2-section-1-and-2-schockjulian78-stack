TrustScan — Details

TrustScan is a dating app safety tool that analyzes message content for scam-like patterns using a lightweight ML classifier plus a rule-based override layer. The goal is a calm, “pause before you trust” second opinion.

Key Features
  • Risk-focused pattern detection (urgent language, financial requests, pressure tactics)
  • One clear verdict with short reasoning
  • Safety-first wording (analyzes the message, not the person)
  • Designed for a simple browser workflow (for now)
Tech / Tools
  • Python + ML model (classifier + vectorizer)
  • FastAPI backend
  • Rule/override layer for clearer results
